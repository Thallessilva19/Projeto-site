Curso de HTML5 com CSS3

HTML (Hypertext Markup Language) é uma linguagem de marcação que permite definir a estrutura e o conteúdo de uma página web. O HTML é composto por um conjunto de elementos, chamados tags, que indicam aos servidores da web como apresentar o conteúdo / CSS é a sigla para Cascading Style Sheets, que significa Folha de Estilo em Cascatas. É uma linguagem de marcação que permite estilizar elementos escritos em HTML, separando o conteúdo da representação visual de um site.
